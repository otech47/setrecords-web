# setmine-web
## Authors
Oscar Lafarga, Quinn Pruitt

## Description
Setmine Web App in React.js

#### Setup the Application
```
$ npm install -g webpack webpack-dev-server
$ npm install
```

#### Running the Server


```
$ npm start
```

#### Running the Application (in a new Terminal tab)

This compiles all assets as well

```
$ webpack-dev-server
```
